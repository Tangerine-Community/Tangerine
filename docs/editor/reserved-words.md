# Reserved words in Tangerine

The following words should not be used as field names in Tangerine because they will cause clashes with mysql export and other features:
- buildChannel
- buildId
- caseDefinitionId
- caseId
- caseRoleId
- caseEventId
- collection
- complete
- dbRevision
- deviceId
- estimate
- eventformid
- formID_sanitized
- groupId
- inactive
- participantId
- required
- startDate
- startUnixtime
- type
- uploadDatetime
